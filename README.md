# Theresa Godfrey Adolf — Tressy Portfolio

Personal portfolio website built with **Next.js** and **TypeScript**.

## Features

- Hero section with introduction
- About me with quick facts
- Education timeline (primary through diploma)
- Skills and professional experience (IT maintenance & networking)
- Contact via WhatsApp and email
- Responsive design with mobile navigation

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18.18 or later (includes npm)

### Install & Run

```bash
cd "Tressy Portfolio"
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Customize

Edit `src/data/portfolio.ts` to update name, education, skills, experience, or contact details.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
